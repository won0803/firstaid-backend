# 🚒 강원소방 수요예측 검증을 위한 구급 출동 데이터 시각화 및 분석 도구

공공기관의 분산된 연도별 소방 출동 데이터를 수집·정제하여 분석용 통합 데이터베이스를 구축하고, 이를 카카오맵 API와 연동하여 출동 데이터를 공간적으로 시각화 및 통계 분석할 수 있도록 구현한 시스템입니다. 대용량 데이터 전처리 파이프라인 구축과 데이터 조회 및 시스템 예외 처리 최적화에 초점을 맞춰 개발했습니다.

---

## 🛠 사용 기술
* Java / Spring Boot / Spring Data JPA
* MySQL
* Python (데이터 전처리 및 CSV 추출 파이프라인)
* Kakao Maps API (출동 데이터 공간 시각화)
  
---

## 🔥 핵심 구현 및 구조적 설계

### 1. 데이터 전처리 및 데이터베이스(DB) 통합 파이프라인
* **재난번호 중심 데이터 병합:** 분산되어 있던 연도별 엑셀 데이터를 재난번호(`DSR_SEQ`)를 핵심 키(Key)로 사용하여 출동 정보와 정확한 좌표 데이터를 하나로 묶어 관리하는 통합 DB 기반을 마련했습니다.
* **공간 데이터 무결성 확보:** 위치 데이터(`GIS_X`, `GIS_Y`)를 `double` 형으로 정밀하게 설정하여 지도 시각화 시 발생할 수 있는 오차를 최소화했습니다. 공공기관 데이터 특유의 복잡한 위치 수치를 카카오맵이 이해할 수 있는 위·경도 좌표로 정확하게 변환하여 사고 지점을 오차 없이 표시합니다.
* **ETL 자동화:** 연도 입력만으로 파일 검색부터 CSV 추출 및 DB 적재까지 일괄 처리하는 파이프라인을 구축하여 데이터 관리 효율성을 극대화했습니다.

### 2. 사용자 중심의 지점 및 격자 상세 분석 통계
* 사용자가 원하는 분석 지점(`start_point_lwc`)을 언제든 자유롭게 추가하고 관리할 수 있도록 독립 테이블로 분리하여 유연한 데이터 구조를 설계했습니다.
* **HTTP 메서드 최적화:** 단순히 데이터를 읽어오는 조회 기능은 `GET` 방식으로 처리하여 데이터 요청의 효율성을 높였고, 정밀한 통계 분석과 지점 추가/삭제 등의 관리 기능은 `POST` 방식을 적용하여 데이터의 안정성과 보안성을 동시에 확보했습니다.

## 🛠 시스템 예외 처리 및 최적화 (Troubleshooting)

### 🚨 데이터 편차로 인한 지점 통계 그래프 시각화 오류 최적화
* **문제 상황:** 특정 지점의 통계 데이터 중 비정상적으로 큰 값(예: 100KM 이상)이 포함될 경우, 차트의 스케일이 왜곡되어 전체 그래프가 빽빽하게 뭉치거나 오류가 난 것처럼 보이는 현상이 발생했습니다.
* **해결 방안:** 데이터 표현 범위를 조정하는 최적화 로직을 도입하여, 100KM 이상의 비정상적인 극단값이 들어오더라도 차트상에서는 최대 20KM까지만 표시되도록 제한(상한선 설정)함으로써 전체 데이터의 시각적 가독성과 그래프 안정성을 확보했습니다.

### 🚨 무효한 조건 선택에 따른 통계 결과의 논리적 오류 차단
* **문제 상황:** 사용자가 지역을 선택하지 않거나, 분석 기간 설정 시 시작일(Start Date)을 종료일(End Date)보다 늦은 날짜로 잘못 지정하는 등의 논리적 입력 오류가 필터링 없이 서버로 요청되는 문제가 있었습니다.
* **해결 방안:** 프론트엔드 및 백엔드 경계에서 빈 항목 검증과 날짜 역전 현상을 사전에 판별하는 '논리적 데이터 필터링'을 구현했습니다. 조건이 유효하지 않을 경우 알림창을 통해 데이터 조회를 선제적으로 차단함으로써, 통계 데이터의 논리적 무결성을 지켰습니다.

### 🚨 필수 항목 누락 방지를 위한 실시간 유효성 검사 및 자동 입력
* **문제 상황:** 사용자가 새로운 분석 지점을 추가할 때, 필수 데이터 입력을 누락하여 발생하는 DB 제약조건 위배 및 서버 에러 위험이 있었습니다.
* **해결 방안:** 지점 추가 폼에 '실시간 유효성 검사'를 적용하여 필수 항목이 누락되면 입력창 아래에 즉각적인 빨간색 경고 메시지를 띄워 실수를 방지하고, 사용자가 내용을 입력하는 즉시 메시지가 사라지도록 설계해 직관적인 UI 피드백을 제공했습니다. 더불어, 좌표 입력 시 해당 좌표에 매칭되는 시(City) 정보가 자동으로 입력되도록 편의 기능을 고도화했습니다.
